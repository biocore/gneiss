Introduction:

Balances are an alternative technique to investigate relationships between proportions. Rather than focusing on proportions, log ratios or balances are investigated in analyses. This technique has many beneficial theoretical properties, while avoiding issues assoicated with compositionality in differential abundance, correlations and regression. More details on balances can be found here.

Installation:

To install this package, it is recommended to use conda. An environment can installed as follows 
conda create -n gneiss_env python=3
gneiss then can be installed as follows 
source activate gneiss_env
conda install pyqt
pip install gneiss
